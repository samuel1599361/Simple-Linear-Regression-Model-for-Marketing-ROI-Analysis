# Marketing ROI Analysis Using Simple Linear Regression
## Project Overview
This project analyzes marketing data using Simple Linear Regression with Python and statsmodels. The goal is to identify which advertising channel (TV, Radio, or Social Media) has the strongest relationship with Sales and provide business recommendations for marketing budget allocation.
## Objectives
- Load and clean the dataset
- Perform exploratory data analysis (EDA)
- Identify the best predictor of Sales
- Build an OLS regression model
- Validate regression assumptions
- Interpret regression outputs
- Recommend the best marketing channel for ROI optimization
